‚ö†Ô∏è Privacy Notice: Hybrid Mode Data Handling

What is sent to cloud services:
  • Extracted email text content only (no attachments, images, or files)
  • Basic metadata like thread length and subject line
What is NEVER sent:
  • Attachment files, images, or documents
  • Personal identifiers or contact details
  • Voice recordings or audio data
When cloud processing occurs:
  • Device models are unavailable or unsupported
  • Content exceeds on-device processing limits (32,000+ characters)
  • Local processing fails due to memory constraints
Your control: You can disable cloud fallback anytime by selecting "On-device only" mode. Processing indicators will show when cloud services are used.

Current Status: Cloud fallback is not yet implemented to maintain this extension's privacy-first design. Hybrid mode selection shows this privacy notice but continues using on-device processing only.
